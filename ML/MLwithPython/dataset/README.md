# Datasets

## Advertising Dataset

- Overview: This dataset contains fabricated information about the features of users responding to online advertisements, including their sex, age, location,daily time spent online, and whether they clicked on the target advertisement.The dataset was created by Udemy course instructor Jose Portilla of Pierian Data and is used in his course Python for Data Science and Machine Learning Bootcamp.
- Features: 10
- Missing values: No
- File name: advertising.csv
- http://scatterplotpress.com/p/datasets

## Melbourne Housing Market Dataset

- Overview: This dataset contains data on house, unit, and townhouse prices in Melbourne, Australia. This dataset comprises data scraped from publicly available real estate listings posted weekly on www.domain.com.au. The full dataset contains 21 variables including address, suburb, land size, number of rooms,price, longitude, latitude, postcode, etc.
- Features: 21
- Contains missing values: Yes
- File name: Melbourne_housing_FULL.csv
- https://www.kaggle.com/anthonypino/melbourne-housing-market/#Melbourne_housing_FULL.csvBerlin
